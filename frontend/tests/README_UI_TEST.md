# UI 自动化测试说明

## 测试脚本

- **测试文件**: `ui-test.js`
- **测试框架**: Playwright
- **浏览器**: Chromium (Headless)

## 运行测试

```bash
cd frontend

# 确保前端开发服务器运行
npm run dev

# 运行测试
node tests/ui-test.js
```

## 测试覆盖

### 登录页面 (5 个测试)
1. 显示登录表单
2. 验证空用户名
3. 验证空密码
4. 成功登录并跳转
5. 失败登录错误处理

### 主布局 (5 个测试)
1. 显示侧边栏和菜单
2. 展开/收起侧边栏
3. 切换菜单页面
4. 显示用户信息
5. 退出登录

## 输出

- **测试报告**: `tests/UI_TEST_REPORT.md`
- **测试截图**: `tests/screenshots/`
