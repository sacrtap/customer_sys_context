## 失败原因分析

### 1. 登录超时 (影响 5 个测试)

**错误日志**:
```
TimeoutError: page.waitForURL: Timeout 10000ms exceeded.
waiting for navigation until "load"
```

**原因**:
- 登录页面输入框 placeholder 可能不是"用户名"和"密码"
- 或者登录成功后不跳转到 `/dashboard`

**解决方案**:
1. 检查登录页面的实际 HTML 结构
2. 使用 DevTools 查看输入框的真实 placeholder
3. 放宽 URL 匹配条件

### 2. 按钮选择器不匹配

**原因**:
- 按钮可能存在但选择器 `.header-actions button` 未匹配
- 或者页面尚未完全渲染

**验证方法**:
```bash
# 手动访问页面检查
curl http://localhost:5173/settlements
```

### 3. 侧边栏响应式异常

**观察**:
```
Desktop 侧边栏：隐藏
Mobile 侧边栏：隐藏
```

**问题**: Desktop 模式下侧边栏应该展开

**可能原因**:
- 主布局组件 CSS 样式问题
- 侧边栏默认状态设置为收起
- 媒体查询阈值设置不当
