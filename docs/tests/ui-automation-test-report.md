## 问题总结

### 前端问题
1. 部分页面导航后数据加载较慢
2. 表格元素在某些情况下未能及时渲染

### 测试脚本问题
1. 部分选择器依赖页面结构，可能因 UI 变化而失效
2. 等待时间设置可能不够充分
